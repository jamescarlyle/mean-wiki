What is Mean Wiki?

Mean Wiki allows you to create a set of personal Wiki pages that link to each other.

Mean Wiki pages are stored within your browser (using "local storage"), allowing you to work offline.

Mean Wiki pages are also stored on a remote server, but protected so that only you can see them. This means that you can access them from a different device, for example your phone or tablet. We're working hard to ensure that it's easy to synchronise changes with the copy of the pages on the server.

Start your Wiki by following these steps:

  1. Enter some text in the sandbox below.
  2. Link to a new page that you wish to create by typing its names using 2 or more words joined together and with the first letter capitalised, like HomePage or ToDoList. This will automatically create a link.
  3. If the new page hasn't yet been edited, a flag will appear next to the name link.
  4. Click on the link, navigate to the page, and enter some content using further name links if more pages are needed.
  5. After you have finished editing the page, click on the Save button and the page will be saved locally and remotely.

Mean Wiki was built using Mongo, Express, Angular.js and Node - hence the acronym that forms part of the name. "Wiki" means "quick" in Hawaiian and the Wiki goal is that it is quick to edit the web.
